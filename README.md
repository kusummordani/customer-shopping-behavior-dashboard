# Customer Shopping Behavior Dashboard

## Project Overview
This project is an interactive Excel dashboard built to analyze customer shopping behavior using Pivot Tables, Pivot Charts, slicers, and formulas.

The dashboard helps identify trends based on:
- Seasonal purchase behavior
- Age group analysis
- Purchase frequency patterns
- Customer distribution by location

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Formulas
- Conditional Formatting

## Dashboard Features
The dashboard includes the following insights:

### 1. Seasonal Purchase Trends
Analyzes how item purchases vary across different seasons.

### 2. Age Group Analysis
Shows customer behavior by age group to identify key customer segments.

### 3. Purchase Frequency Analysis
Examines how often customers purchase different items.

### 4. Customer Distribution by Location
Highlights the number of customers across different locations.

### 5. Interactive Slicer
A **Season slicer** is included to make the dashboard interactive and allow dynamic filtering.

## Key Insights
- Customer purchase behavior changes across different seasons.
- Some item categories perform better in specific seasonal periods.
- Customer distribution varies by location.
- Purchase frequency patterns help identify shopping habits.

## Project Objective
The goal of this project was to practice Excel data analysis and dashboard creation by transforming raw customer shopping data into meaningful business insights.
